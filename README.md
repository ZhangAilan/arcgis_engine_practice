## 功能介绍

### 文件操作

1. **读取和写入shp文件**
2. **读取和处理栅格数据**
3. **处理矢量数据**
4. **创建和操作TIN数据**
5. **处理CAD文件**
6. **读取和保存MXD文档**

### 地图操作

1. **拉伸**
2. **缩放**
3. **绘制点、线、面**
4. **移动地图**
5. **保存地图文档**
